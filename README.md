# Quantity Measurement App

##  Overview

The Quantity Measurement App is a use-case-driven application built using Core Java and Maven.  
This project is designed to implement quantity measurement operations step by step, following clean coding principles, object-oriented design, and proper Git branching strategy.

Each functionality is implemented as a separate Use Case (UC).

---

##  Project Structure

- Main Class: `QuantityMeasurementApp`
    - Contains actual application logic
- Test Class: `QuantityMeasurementAppTest`
    - Contains all JUnit test cases
